# 后日谈系统

> 创建日期：2026-01-17
> 文件位置：`src/赵霞/脚本/游戏逻辑/afterStorySystem.ts`
> 状态：已实现

## 概述

后日谈是游戏结局触发后的"奖励模式"，让玩家体验结局后的日常。

**核心设计**：
- 时长：2轮对话
- 触发：结局完成后（后日谈已解锁标记为true）
- 完成后：解锁自由模式，可无限制继续对话

---

## 支持的结局类型

| 结局类型 | 后日谈支持 | 苏文状态 | 打断系统 |
|---------|-----------|---------|----------|
| 完美真爱结局 | ✅ | 主动成全，成为苦主 | 关闭 (0) |
| 真好结局 | ✅ | 住院中（头孢+酒反应） | 关闭 (0) |
| 假好结局 | ✅ | 在家，怀疑中 | 根据行为判定（待设计） |
| 坏结局 | ❌ | - | - |
| 普通结局 | ❌ | - | - |
| 混乱结局 | ❌ | - | - |

---

## 触发条件

```typescript
function shouldTriggerAfterStory(data: SchemaType): boolean {
  // 必须有好结局（完美真爱、真好、假好）
  const type = getAfterStoryType(data);
  if (!type) return false;

  // 后日谈已解锁
  if (!data.结局数据.后日谈已解锁) return false;

  // 后日谈未触发
  if (data.结局数据.后日谈?.已触发) return false;

  return true;
}
```

## 数据结构

```typescript
interface AfterStoryState {
  已触发: boolean;      // 后日谈是否已开始
  当前轮数: number;     // 当前第几轮 (1 或 2)
  已完成: boolean;      // 2轮是否全部完成
  自由模式: boolean;    // 是否进入自由模式
}

type AfterStoryType = '完美真爱' | '真好结局' | '假好结局' | null;
```

## 完美真爱结局后日谈

### 苏文设定
- 他已经知道妻子和儿子的关系
- 他选择了接受，甚至发展出了特殊癖好
- 他愿意在旁边服务，表现出服从
- 打断系统：**关闭**

### 赵霞状态
- 完全沦陷，对玩家有绝对的依赖和爱
- 在苏文面前也不再掩饰
- 可能会当着苏文的面和玩家亲密

### 场景设计

**第1轮：清晨的日常**
```
清晨的阳光透过窗帘洒进房间。
赵霞靠在{{user}}的肩上，脸上带着满足的笑容。
苏文站在门口，手里端着早餐托盘，表情复杂但平静。
```

**第2轮：苏文与录像**
```
晚上，{{user}}路过客厅，发现苏文独自坐在沙发上。
电视屏幕上播放着什么...
定睛一看——是婚礼的录像。
```

- "是他自己不小心的"

**第2轮：医院探视**

赵霞例行去医院探视苏文。

## 状态栏日期显示

### 设计理念

结局完成后，状态栏不再显示游戏内的"Day X"倒计时，而是显示玩家电脑的真实日期。

**含义**：赵霞已经"逃出了循环"，进入了玩家的现实世界。

### 显示切换

| 阶段 | 状态标题 | 日期显示 |
|------|---------|---------|
| 游戏中 | 根据阶段变化 | Day X 剩余X天 |
| 后日谈进行中 | 【后日谈 1/2】完美真爱结局 | 真实日期 |
| 自由模式 | 【自由模式】完美真爱结局 | 真实日期 |

### 技术实现

使用酒馆内置宏变量：

```typescript
function generateDateDisplay(data: SchemaType): string {
  if (shouldShowRealDate(data)) {
    // 使用酒馆宏显示真实日期
    return '{{datetimeformat::YYYY年M月D日}}';
  } else {
    // 游戏中显示游戏日期
    const day = data.世界.当前天数;
    const remaining = Math.max(0, 5 - day);
    return `Day ${day} 剩余${remaining}天`;
  }
}
```

## 自由模式

后日谈2轮完成后，自动进入自由模式：

- 无时间限制
- 无结局判定
- 保持结局时的打断系统强度
- 可无限制继续与赵霞互动

```typescript
function advanceAfterStoryRound(data: SchemaType): void {
  const currentRound = data.结局数据.后日谈.当前轮数;

  if (currentRound >= 2) {
    // 完成后日谈，进入自由模式
    data.结局数据.后日谈.已完成 = true;
    data.结局数据.后日谈.自由模式 = true;
  } else {
    data.结局数据.后日谈.当前轮数 = currentRound + 1;
  }
}
```

## 与其他系统的交互

1. **结局系统**：结局完成后标记 `后日谈已解锁 = true`
2. **状态栏系统**：根据模式切换日期显示格式
3. **境界打断系统**：根据结局类型调整打断强度
4. **提示词注入系统**：注入后日谈/自由模式的AI引导
