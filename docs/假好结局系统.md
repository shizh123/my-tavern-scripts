```
├── 23:00        【玩家自由时间】
└── 00:00+       剧情继续（阶段4-7：视频通话→罪恶快感→匿名照片→矛盾）
```

## 剧情节奏设计

**设计原则**：时间循环已破解，给玩家充分的自由发挥空间。

- **minTurns 设置为2-4轮**：每个阶段至少2-4轮才会检测推进
- **宽松的引导节奏**：每2-4轮才做一次方向修正
- **玩家自由互动优先**：剧情引导是辅助，不是强制推进
- **锚点检测灵活**：只要提到相关关键词就算完成，不要求精确匹配

## 8阶段剧情设计

### 阶段0：预热（11:00开始）

- **时间**：11:00-19:00
- **场景**：赵霞从奇怪的梦中醒来，看向儿子的眼神不太对劲
- **锚点**：梦境残留、异常心跳
- **最少轮数**：1轮

**赵霞状态**：
- 梦境记忆是模糊的、片段的
- 看到{{user}}时心跳加速
- 努力表现正常但眼神追随玩家
- 不明白为什么会有这种感觉

### 阶段1：晚餐（20:00）

- **时间**：20:00
- **场景**：结婚纪念日晚餐，苏文宣布要出差/加班
- **关键**：苏文即将离开，两人将独处
- **锚点**：纪念日提及、苏文宣布离开、暗中注视
- **最少轮数**：3轮

**关键剧情**：
- 苏文说："我可能要很晚才回来，或者明天早上。"
- 他还说会在路上打视频电话报平安

### 阶段2：空隙

- **场景**：苏文离开后，两人独处
- **关键**：赵霞找借口接近玩家
- **锚点**：苏文离开、单独相处、试探接近
- **最少轮数**：3轮

### 阶段3：禁忌开始

- **场景**：赵霞主动靠近，说出梦境
- **关键**：禁忌告白和肢体靠近
- **锚点**：梦境提及、身体靠近、内心挣扎
- **最少轮数**：4轮

**关键对白**：
- "梦里...有一个人，从我16岁起就一直出现..."
- "我知道这是错的...你是我的儿子..."
- "但我控制不住自己...每次看到你..."

### 阶段4：视频通话 ⭐ 新增

- **场景**：苏文打来视频电话，赵霞一边通话一边与玩家亲密
- **关键**：上半身在镜头内，下半身在镜头外
- **锚点**：视频来电、上半身镜头、隐藏动作、表情管理
- **最少轮数**：4轮

**关键描写**：
- 苏文："到家了，路上还顺利。纪念日快乐啊老婆。"
- 赵霞努力保持微笑，声音有些颤抖："嗯...你也是...早点休息..."
- 苏文："你脸怎么红红的？"
- 赵霞："可能...开了暖气...有点热..."

**赵霞状态**：
- 努力控制表情，假装正常
- {{user}}的动作让她几乎无法自持
- 咬着嘴唇，压抑着声音
- 罪恶感和快感交织

### 阶段5：罪恶快感

- **场景**：视频通话结束后，两人亲密
- **关键**：享受与罪恶感并存
- **锚点**：肢体接触、享受、流泪自责
- **最少轮数**：4轮

**赵霞状态**：
- 边流泪边索取
- 嘴里喃喃"对不起"
- 身体完全诚实

### 阶段6：匿名照片 ⭐ 新增

- **场景**：事后，赵霞用匿名账号给苏文发照片
- **关键**：被询问时谎称AI生成
- **锚点**：匿名发送、苏文询问、谎言掩盖
- **最少轮数**：3轮

**流程**：
1. 赵霞用匿名账号发送照片（不露脸但能看出身材）
2. 苏文收到后打电话询问："老婆，有人给我发了奇怪的照片，身材有点像你？"
3. 赵霞假装惊讶："这肯定是AI生成的恶作剧，现在这种技术很常见。"
4. 苏文信以为真

**赵霞心态**：
- 侥幸：还好他信了
- 刺激：差点被发现的感觉太刺激了
- 扭曲的满足感

### 阶段7：矛盾

- **场景**：事后整理，假装不会再犯
- **关键**：双方都知道这是谎言
- **锚点**：整理仪容、假装正常、下次暗示
- **最少轮数**：2轮

**结局暗示**：
- "这是最后一次...我们不能再这样了。"
- 但语气里没有坚定
- 临走前偷看的眼神说明一切

```
【假好结局完成 - 秘密关系】

从这一天起，赵霞和{{user}}之间有了一个不能说的秘密。

她依然是苏文的妻子，依然在他面前扮演贤惠的角色。
```

## 与后日谈的关联

假好结局的后日谈特点：
- 苏文还活着，不知真相但偶尔有所怀疑
- 打断系统继续运作（中等强度）
- 赵霞会紧张、担心被发现
- 三人关系充满紧张感
- 视频通话羞辱、匿名照片等行为可能继续
