## 一、已确定的大方向

1. **苏文 = 威胁（不变）**
   - 保持丈夫随时可能撞破的威胁定位，紧张感是卖点。
   - 只修复「突脸太频繁/不可预测/能ROLL刷」让威胁变公平，不把苏文变成可调教/堕落对象。
   - （旧设计稿里的「疑心值→绿帽值双段制」已作废。）

2. **念头植入 = 秦璐的核心玩法与护城河**
   - 不能丢，要**深耕**。这是秦璐区别于其他卡的根。

3. **道具 = 网店即时获取**
   - 网店界面浏览→下单→立即到手→立即用，无需等待时间。
   - 作用两类：①规避风险（对抗苏文）②加速调教。
   - 是否需要货币：**待定**（作者还没想好；若引入也只用单一货币变量）。

4. **多结局 = 必须支持**
   - 决定因素待定。

5. **两条贯穿全局的铁律**
   - **数据隔离（最高优先级）**：AI 只看当前阶段/场景需要的数据。后期 token 膨胀会让 AI 注意力下降，大量无关数值会霸占注意力导致胡言乱语。系统全知，AI 按需知情。
   - **数值精简**：变量越少越好，派生值不存独立字段。

## 二、作者从云霜凝吸取的教训（重置秦璐要避开的）

- **线性 = 死。** 云霜凝问题：作者预设结局，玩家被特殊剧情牵着走完轨道到达预设终点。
  - 酒馆卡灵魂是「无限可能」，被锁成一条轨道是弊端。
  - 玩家通关一次就没动力再玩。
  - 边做边发 → 玩家中途就猜到结局，等结局版出来热情已凉。
  - 边做边发还带来大量 BUG，体验差 + 已知结局 = 没人想重开。

- **秦璐要走云霜凝的反面**：作者搭系统，玩家用念头植入自己玩出千差万别的过程和结局。可重复游玩性来自**玩家的塑造自由**，不是作者写更多分支剧情。

## 三、特殊剧情 / 奖励的设计反思（重点）

- 云霜凝的「特殊剧情」本意是奖励，实际**变味成必经关卡**。
  - 原因：它卡在主线推进路径上 → 必经的东西就不是奖励而是任务。
  - 奖励的本质 = 可有可无、锦上添花、玩家自己挣来的惊喜、不看也能继续玩。

- **秦璐想要的「真正的奖励」**：
  - 挂在**念头植入这套核心玩法**上。
  - 玩家把念头种成功、养成习惯、推进到一定阶段 → 系统掏出特殊剧情作为犒赏。
  - 不在主线路径上 → 不会变关卡。
  - 绑在念头玩法上 → 玩家为拿奖励更深度玩核心玩法 → 深耕念头。
  - 由玩家塑造决定触发哪些 → 千人千面 → 支撑无限可能与重复游玩。
  - 和结局解耦 → 看不看奖励都能继续玩。

- **触发逻辑：作者选了 B（念头组合式）**，但「目前不知道如何表现」。
  - B = 奖励长在玩家植入的**具体念头/习惯组合**上，而非阶段里程碑（A）。

### 针对 B「如何表现」我（助手）提出的破局思路（待作者回来确认）

- 难点不在剧情，在**「识别」**：系统怎么知道玩家养成了能触发奖励的组合。
- **破局：不识别「具体念头文字」，识别「念头的性质标签」**（秦璐念头系统已有分类：陪伴/肢体/暴露/性行为/身份认同…）。
- 奖励绑在「某类标签累积够」或「某几类标签的搭配」上，而非某条具体念头。
- 三种表现形式（可叠用）：
  1. **倾向养成（朴素）**：某类念头反复植入累积够 → 解锁对应倾向的奖励剧情。
  2. **人格画像（进阶）**：系统维护秦璐「被塑造成什么样的人」的画像（依赖型/放荡型/母性型/奴性型…），同阶段不同画像掏出不同奖励 → 千人千面。
  3. **组合解锁（隐藏感最强）**：两类标签特定搭配才触发隐藏奖励（如暴露+寝取倾向 → 冒险剧情），制造探索惊喜 → 想重开试别的组合。
- 奖励体量（A小段/B一场戏/C都有）—— **未确认，被下面更优先的问题打断。**

## 四、被提到的更底层问题（比突脸更根本，待续）⭐

作者指出：**秦璐在念头还没成熟为习惯时，角色已经被严重影响；模型根本分不清「念头的表现」和「习惯的表现」。**

助手分析（待作者回来确认这个区分）：

- 根因：**没有定义「念头期」和「习惯期」在叙事表现上的差异**，没把差异明确注入给模型，模型只能糊成一团 → 念头一植入角色就直接变了，阶段感全没。
- 建议的区分：
  - **念头期（培育中）= 心理层面的暗流**：秦璐**不该照做**，甚至不知道自己有这念头。表现为一闪而过的杂念、莫名冲动、困惑「我怎么会这么想」、抗拒、压下去。是「动摇」不是「改变」。
  - **习惯期（已成熟）= 行为层面的事实**：自然、不假思索地去做，无心理障碍。才真正影响行为。
- 与作者目标的咬合：
  - 念头期的「内心挣扎/暗流」本身是被浪费的巨大玩法空间和情绪张力 → 正好用来**深耕念头玩法**。
  - 念头**熬成习惯的那一刻** = 天然的奖励时刻（种子发芽）。
- 关联「不同模型表现力」：弱模型理解力差，若不把「念头期压抑 / 习惯期放开」写成清晰强制的行为指令注入（而非指望模型自悟），必然糊；写清楚了弱模型也能照演。

### 待作者回来回答的关键问题

> 你认同「**念头期 = 内心暗流/不自觉/会抗拒，习惯期 = 自然行为/无障碍**」这个区分吗？还是对这两个阶段的表现有别的设想？

这个定了，「念头怎么深耕」「奖励挂在哪」才有地基。
