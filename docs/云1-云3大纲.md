# 云1-云3 故事大纲

## 世界观设定

### 修炼境界（从低到高）
筑基 → 结丹 → 元婴 → 出窍 → 羽化 → 地仙境 → 仙界

### 地理
- 乾北大陆极北之地
- 天极山脉 / 天霜峰 — 寒霜门驻地，终年严寒但山顶反常温暖

### 核心设定
- **双修**：男女修士交合可互相提升修为，女修越动情效果越强
- **纯阳童子之身**：主角特殊体质，纯阳灵力可克制寒毒
- **极阴之体**：极稀有体质，修炼冰系功法如虎添翼，但极难动情
- **天霜决**：寒霜门老祖留下的冰系顶级功法，仅主脉可修

| 角色 | 身份 | 特点 |
|------|------|------|
| **李普** | 天一门掌门虚灵子大弟子，约15岁 | 纯阳童子之身，金丹后期，外貌定格童子模样 |
| **云霜凝** | 寒霜门掌门，白发白睫 | 极阴之体，修天霜决，羽化后期（实为地仙境） |
| **苗广** | 云霜凝之夫，儒雅中年 | 羽化初期，与虚灵子为旧识 |
| **苗喧** | 苗广与云霜凝之子，年轻 | 结丹期，初期傲慢，后转变态度 |
| **兰莲儿** | 寒霜门弟子 | 筑基后期，负责护送李普 |

## 三章情节梗概

### 云1 — 相遇篇
- 李普乘飞舟三日抵达天极山脉，寒霜门六名女弟子随行护送（皆知其纯阳之身）
- 苗广在码头迎接，其子苗喧出言轻蔑被当场打脸
- **核心背景揭露**：云霜凝三百年前踏入羽化期，近日冲击地仙境失败，被"地劫寒毒"侵蚀神魂，沉睡如植物人，意识清醒但无法控制肉体
- 苗广请求李普以纯阳灵力与云霜凝双修，阴阳调和以修复被寒毒侵蚀的神魂
- 李普通过触碰与沉睡的云霜凝在神魂空间内交流
- **云霜凝自述**：天生"性冷淡"（无法动情），与夫君双修效果极低，故李普提出先设法唤醒其感觉
- 苗广退出门外守候，"治疗"开始

### 云2 — 调教篇
- 第一日治疗效果显著：云霜凝被完全"攻陷"，由性冷淡到高潮连续
- 苗广在门外发现残留迹象，内心经历：震惊→愤怒→压制→逐渐习惯→主动偷窥
- **苗广的心理转变**：他悄悄将床帐改造为单向透视，每日隐身进屋观看，并由此获得前所未有的强烈刺激
- 李普洞察苗广的行为，有意放大场面以测试其反应
- 云霜凝在调教过程中逐步从清冷神圣转变，寒毒明显消退
- 李普同时给兰莲儿承诺事后帮助其突破筑基瓶颈

### 云3 — 揭幕篇
- **关键反转**：云霜凝恢复部分自主意识，向李普透露她其实已经突破地仙境——冲击时成功了却被寒毒入侵，自己未察觉
- 云霜凝能够施展**千晶幻术**（寒霜门秘传幻法），将苗广导入幻境
- 李普与云霜凝"联手"，瞒着苗广继续"治疗"
- 寒霜门各长老闻讯前来探望云霜凝苏醒，李普以体力耗尽为由让众人离开
- 苗喧向李普正式道歉，两人关系缓和
- 苗广承诺开放宝库，赠予李普突破元婴的天材地宝
- **结尾状态**：云霜凝神魂修复接近完成，等神魂彻底康复即正式踏入地仙境；李普突破元婴之机近在眼前

## 核心冲突与主题

1. **纯阳克极阴**：体质相克相生，双修是治疗手段也是剧情驱动
2. **寒霜门门规**：道侣间从一而终（源自老祖被渣男始乱终弃），形成道德张力
3. **苗广的心态变化**：从护夫到习惯再到欲罢不能的复杂心理弧线
4. **权力与伪装**：李普以弱者姿态示人，实力远超外表，步步主导局面
5. **突破契机**：李普在极阴之地感悟阴阳调和，为元婴突破蓄力
