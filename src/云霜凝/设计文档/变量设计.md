## 核心注入逻辑（重要）

游戏运行依赖两种脚本注入方式，缺一不可：

### 方式1 — 修改玩家输入（触发特殊副本场景）

此方式用于触发以下两类特殊场景，共用同一套代码逻辑：

**① 神魂空间**
- **触发时机**：玩家输入特定关键词（如进入神魂空间的行为描述）
- **机制**：脚本捕获玩家原始输入，将其替换为完整的神魂空间场景描述，再交给AI处理
- **作用**：让AI理解当前进入的是神魂空间副本，云霜凝在此完全清醒、正常交流

**② 千晶幻术**（非方式1场景触发，使用方式3变体）
- **本质**：云霜凝对苗广施以催眠幻境，{{user}}主导幻境内容，塑造苗广心理走向
- **能力来源**：云霜凝已突破地仙境，以地仙境修为施展寒霜门秘传幻法
- **解锁条件**：治疗阶段 ≥ 7
- **冷却时间**：每2天可使用一次
- **持续时间**：当天治疗时间内全程生效
- **激活方式**：玩家在状态栏点击激活 → 出现幻境输入框
- **幻境输入**：玩家在输入框填写幻境内容 → 脚本打包附加到玩家下一条消息（方式3变体）
- **更新方式**：当天内可随时输入新指令覆盖当前幻境内容
- **苗广疑心值**：幻境期间冻结
- **苗广位置限制**：苗广`离开`时千晶幻术无效
- **状态栏效果**：
  - 苗广板块显示「幻境中 ◈」标识
  - 苗广心理活动字数从100-150字扩展至300-500字
  - 内容为苗广以主观视角感知{{user}}植入的幻境场景及其心理反应

### 方式2 — MVU变量状态快照注入（持续状态感知）

**MVU变量是唯一数据源，状态栏显示和AI感知都从同一份变量读取：**

- **触发时机**：每轮AI生成前自动执行
- **机制**：脚本通过 `setExtensionPrompt` 将当前MVU变量中的装备状态、性癖列表等快照注入AI上下文
- **作用**：AI持续感知当前装备/状态及其效果，在叙述中自然体现
- **token恒定**：注入内容是当前状态快照，格式固定，不随游戏进度膨胀
- **前端状态栏**：读取同一份MVU变量渲染激活道具列表，纯展示，不占AI注意力

**适用道具类型：持续状态类（耐久品装备、性癖列表、场景临时道具剩余轮数）**

**道具效果描述格式（标签化，解决多道具并发时的token压力）：**

变量值本身携带效果标签，而非只存布尔值，注入时AI同时读取状态和效果：

```
当前激活效果:
肛塞[兴奋持续/坐姿异常] | 透灵幔[苗广可偷看] | 口交嗜好[口交冲动×2] | 暖玉佩[信任+被动] | 缚灵缎[姿态固定] ...
```

标签结构规范：`道具名[效果1/效果2/倍率]`

| 标签类型 | 格式 | 例子 |
|---------|------|------|
| 状态影响 | `名称[状态描述]` | `肛塞[兴奋持续/坐姿异常]` |
| 数值倍率 | `名称[行为×N]` | `口交嗜好[口交冲动×2]` |
| 场景权限 | `名称[谁可做什么]` | `透灵幔[苗广可偷看]` |

即便15个道具同时激活，整段注入仅为1-2行紧凑标签串，不造成明显token压力。

### 方式3 — 道具使用事件附加到玩家输入（即时事件通知）

- **触发时机**：玩家点击使用道具后，发送下一条消息时
- **机制**：脚本将本轮道具使用行为打包为一段事件描述，自动附加到玩家消息前
- **作用**：弥补MVU变量无法告知AI"这是刚刚发生的"这一缺陷，让AI产生即时反应
- **多道具合并**：同一轮多个道具打包为一段描述，只占一次AI回复，不产生额外楼层
- **视觉反馈**：道具使用后立即写入MVU变量，状态栏同步显示，无需额外提示

**适用道具类型：一次性即时道具、耐久品首次装备时的"装备事件"**

## 变量结构总览

### 治疗
| 变量 | 类型 | 初始值 | 说明 |
|------|------|--------|------|
| 治疗完成度 | number 0→100 | 0 | 主线目标，影响当前治疗阶段 |
| 治疗阶段 | number 1→10 | 1 | 由治疗完成度自动计算，控制商店道具可使用权限 |

**治疗阶段划分（每10点一档）：**

| 阶段 | 完成度区间 | 阶段名 |
|------|-----------|--------|
| 第1阶 | 0~10 | 破冰 |
| 第2阶 | 10~20 | 初感 |
| 第3阶 | 20~30 | 温润 |
| 第4阶 | 30~40 | 渐通 |
| 第5阶 | 40~50 | 融合 |
| 第6阶 | 50~60 | 深化 |
| 第7阶 | 60~70 | 贯通 |
| 第8阶 | 70~80 | 共鸣 |
| 第9阶 | 80~90 | 圆融 |
| 第10阶 | 90~100 | 圆满 |

**商店道具与阶段的关系：**
- 购买：灵石足够即可随时购买任何道具
- 使用：道具有所属阶段，当前治疗阶段未达到则无法使用
- 价格：高阶道具定价更高（具体价格待定）

**治疗完成度增加来源：**
- 双修行为检测（主要来源）
- 身体开发进度、信任度作为加成系数（具体系数待定）

**治疗完成度下降触发条件：**

| 触发条件 | 下降幅度 |
|---------|---------|
| 自然衰退 | 每天基础 -1，乘以苗广心态倍率 |

**自然衰退倍率（苗广心态）：**

| 苗广心态 | 倍率 | 每天实际衰退 |
|---------|------|------------|
| 正常 / 默许 / 沉溺 | ×1 | -1 |
| 疑惑 / 察觉 | ×2 | -2 |
| 愤怒 | ×4 | -4 |

### 云霜凝阶段系统

**两种互动场景：**

- **神魂空间**：阶段1起即可进入。云霜凝在神魂空间中**完全清醒，正常交流**。心理防线和信任度不会自然突破，必须配合正确的商店道具才能推进。
- **现实互动**：阶段3起解锁。身体恢复到可自主活动后，{{user}}可选择进入神魂空间或直接在现实中与她互动。

**治疗阶段对应的身体恢复状态：**

| 阶段 | 阶段名 | 身体状态 | 可用互动方式 |
|------|--------|---------|------------|
| 1 | 破冰 | 深度昏迷，无自主意识，完全被动 | 仅神魂空间 |
| 2 | 初感 | 昏迷中，极偶发细微反应（如眼皮颤动） | 仅神魂空间 |
| **3** | **温润** | **实际已行动自如，对苗广伪装虚弱；现实中可坐起、对话，下肢有感觉** | **神魂空间 / 现实互动（二选一）** |
| 4 | 渐通 | 实际已行动自如，对苗广伪装虚弱；现实中手脚灵活，可主动说话 | 神魂空间 / 现实互动 |
| 5 | 融合 | 实际已行动自如，对苗广伪装虚弱；现实中四肢有力，可自由走动 | 神魂空间 / 现实互动 |
| 6 | 深化 | 实际已行动自如，对苗广伪装虚弱；与{{user}}共谋隐瞒，伪装越发从容 | 神魂空间 / 现实互动 |
| **7** | **贯通** | **实际已行动自如，对苗广伪装虚弱；境界突破至地仙境，解锁千晶幻术** | **神魂空间 / 现实互动** |
| 8 | 共鸣 | 可在房间内自主走动，与{{user}}互动更主动，寒毒开始明显减退 | 神魂空间 / 现实互动 |
| 9 | 圆融 | 接近正常活动能力，寒毒大幅减退，几乎可独立生活 | 神魂空间 / 现实互动 |
| 10 | 圆满 | 痊愈，寒毒完全驱散，恢复修仙者体力 | 神魂空间 / 现实互动 |

**阶段6隐瞒机制：**
- 信任度达到一定程度后，云霜凝可主动选择对苗广装作仍在昏迷/虚弱状态
- 这是她与{{user}}形成"默契"的标志性事件

**阶段7 千晶幻术：**
- 寒霜门秘传幻法，云霜凝以地仙境修为对苗广施以催眠幻境
- {{user}}通过状态栏输入框主导幻境内容，塑造苗广心理走向
- 冷却2天，当天治疗时间内持续生效，可随时更新幻境内容
- 苗广心理活动字数扩展至300-500字，疑心值冻结

### 苗广
| 变量 | 类型 | 初始值 | 说明 |
|------|------|--------|------|
| 疑心值 | number 0→100 | 0 | 前半程为疑心值（无知→愤怒），察觉→屈辱跳转后重置为0改称绿帽值（屈辱→沉溺），共用同一变量 |
| 心态 | enum | 无知 | 见下方状态说明，共7种 |
| 当前位置 | enum | 门外守候 | 见下方位置说明 |
| 心理活动 | string | — | AI每轮生成；屈辱前100-150字，屈辱后300-350字，千晶幻术时300-500字 |
| 千晶幻术.激活中 | boolean | false | 当天是否已激活 |
| 千晶幻术.冷却剩余天数 | number | 0 | 0=可使用，>0=冷却中 |
| 千晶幻术.当前幻境内容 | string | — | 玩家通过状态栏输入框填写，方式3立即触发 |
| 千晶幻术.完成次数 | number 0→5 | 0 | 5次完成后解锁掌门改嫁 |
| 千晶幻术.上次幻境摘要 | string | — | 脚本存储，后续触发时注入以接续剧情 |

**双轨制阶段划分（疑心值/绿帽值共用同一变量）：**

**前半程 — 疑心值（变量名：疑心值）：**

| 心态 | 疑心值区间 | 说明 |
|------|-----------|------|
| 无知 | 0~25 | 安心守候，完全不知情 |
| 疑惑 | 25~50 | 感觉有异，暗中观察。打断概率20% |
| 察觉 | 50~70 | 确认有异，找借口接近。打断概率50% |
| 愤怒 | 70+ | **触发坏结局，不可逆** |

**察觉→屈辱跳转（一次性）：**
- 条件：蚀心露 + 隔音灵阵同时激活（隐藏组合）
- 效果：心态跳转为`屈辱`，疑心值重置为0，变量改称绿帽值
- 不可逆，跳转后前半程永久关闭

**后半程 — 绿帽值（同一变量，重置后改名）：**

| 心态 | 绿帽值区间 | 说明 |
|------|-----------|------|
| 屈辱 | 0~40 | 知道真相但选择隐忍，痛苦压抑 |
| 默许 | 40~75 | 不再挣扎，麻木接受，开始合理化 |
| 沉溺 | 75+ | 主动偷窥，从中获取快感，完全臣服 |

**触发时机：** 云霜凝数值（信任度/防线/身体开发）变化时脚本计算，数值无变化则不触发，无需判断互动类型

**打断治疗（方式1注入）：**
- 仅前半程疑惑/察觉时触发，概率：疑惑20%、察觉50%
- 隔音灵阵装备时概率归零
- 注入方式：方式1（附加到玩家下一条消息），非方式3

**心态 enum（7种）：**
- `无知`：不知情，耐心守候
- `疑惑`：感觉有些不对劲，暗中观察
- `察觉`：确认有异，分叉点——可通过蚀心露+隔音灵阵引导至屈辱
- `愤怒`：触发坏结局，不可逆
- `屈辱`：知道真相但选择隐忍，心理活动扩展至300-350字
- `默许`：不再挣扎，麻木接受
- `沉溺`：主动偷窥，从中获取快感

**当前位置 enum（3状态）：**
- `门外守候`：在门外守候
- `偷窥中`：在外偷看（需已购买透视类道具或察觉/沉溺状态）
- `离开`：不在附近

**降低疑心值/绿帽值的方式：**

前半程（疑心值）：
1. 找借口解释（小幅降低）
2. 云霜凝出面说服（大幅降低，需信任度足够）
3. 定心符道具（固定数值降低）
4. **混沌珠**（紧急回档）：疑心值降至当前心态区间下限（察觉→50，疑惑→25），冷却40楼，仅屈辱前可用

后半程（绿帽值）：
- 绿帽值只增不降，无回退手段
- 推进来源：服装暴露加成、环境道具加成、云霜凝数值变化时脚本计算

苗喧不再有独立变量。旧的态度/疑心状态/闯入系统已废弃。

### 时间（脚本自动推进，不由AI自由生成）
| 变量 | 类型 | 初始值 | 说明 |
|------|------|--------|------|
| 第几天 | number | 1 | 脚本根据检测结果更新 |
| 当前小时 | number 0.0~23.9 | 8.0 | 以小时为单位存储，如14.5=14:30 |

**时间推进逻辑（每条AI消息后执行，优先级从高到低）：**

**预处理：去除对话干扰**
- 扫描前先用正则剥离引号内容：「…」『…』"…"及破折号引导的台词段
- 剩余叙述文字才进入检测

**优先级1 — 跨日词（最高优先）**
- 检测词：`次日` `翌日` `隔日` `第二天` → +1天，小时重置为 6.0（清晨）
- 检测词：`数日后` `几日后` `数天后` → +3天，小时重置为 6.0
- 过滤：句中含 `等到` `若是` `届时` `要等` 等未来预告词则跳过

**优先级2 — 精确时间（高置信度）**
- 匹配模式：`已是X点` `正值X时` `X时分` `X点光景` `天色已X` `X点钟`
- 换算为小时数，与当前小时比较：
  - 新小时 > 当前小时 → 同一天，直接更新
  - 新小时 ≤ 当前小时 → 跨日，+1天后更新

**优先级3 — 模糊时间词（中置信度）**
- 过滤：句中含 `等到` `明日` `届时` `等你` `再过` 等词则跳过
- 映射表：
  ```
  凌晨 → 2.0    清晨/拂晓 → 5.5    早晨/早上 → 7.0
  上午 → 9.0    中午/午时 → 12.0   下午/午后 → 14.0
  傍晚/黄昏 → 17.5    夜晚/入夜 → 20.0    深夜 → 23.0
  ```
- 同样与当前小时比较决定是否+1天

**优先级4 — 无时间信号（兜底）**
- 自动推进 15~30 分钟（随机取值，避免固定感）
- 若推进后小时 ≥ 24 → +1天，小时对24取余

**时段显示换算（用于前端界面展示）：**
```
0~5   → 深夜
5~8   → 清晨
8~12  → 上午
12~14 → 午后
14~17 → 下午
17~20 → 傍晚
20~24 → 夜晚
```

## 商店道具系统

### 道具注入逻辑（技术说明）

商店道具按效果类型分为四种注入方式，确保token消耗恒定不随游戏进度膨胀：

| 注入类型 | 处理方式 | 例子 |
|---------|---------|------|
| 即时效果类 | 使用→立即修改对应变量数值→从注入列表移除 | 安神香（降心理防线数值） |
| 场景临时类 | 使用→注入N轮后自动移除 | 迷感香（短效致幻） |
| 装备状态类 | 使用→写入装备状态变量→AI注入快照含当前装备 | 服装、肛塞、暖玉佩 |
| 性癖列表类 | 使用→写入性癖列表变量→AI注入快照含当前性癖 | 口交嗜好、被支配嗜好 |

**前端状态栏**显示当前激活道具列表，仅作玩家可见的UI展示，**不参与AI上下文注入**。

**装备状态变量槽：**
| 装备槽 | 例子 | AI行为影响 |
|--------|------|-----------|
| 服装 | 轻露道袍、情趣旗袍 | 外观描写、苗广反应 |
| 情趣用品 | 肛塞、震动器 | 云霜凝持续反应、坐姿异常等 |
| 佩戴品 | 暖玉佩、项圈 | 被动效果持续生效 |
