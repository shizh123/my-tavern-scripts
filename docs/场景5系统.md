# 场景5线性剧情系统

> 创建日期：2026-01-17
> 文件位置：`src/赵霞/脚本/游戏逻辑/scene5System.ts`
> 状态：已实现

## 概述

场景5是通过安眠药触发的特殊关卡，采用12步线性剧情设计。核心主题是"婚礼的改写"——玩家在赵霞25岁结婚当天的记忆中出现，让她对婚姻产生动摇。

---

## 核心机制

| 机制 | 说明 |
|------|------|
| 12步线性剧情 | 从婚礼前到摘下戒指的完整流程 |
| 时间决定步骤数 | 07:00进入=13步可用，19:00进入=1步 |
| 玩家意图分析 | 分析态度/行为/目标，生成定向AI引导 |
| 契合度系统 | 高契合+10%，低契合+5% |
| 多次进入机制 | 第1次步骤1-10，第2次步骤11-12+自由互动 |
| 强制退出 | 19:00触发退出描写，20:00切换日常 |

## 阶段划分

```
1-2步 [初入]  - 化妆间相遇，建立存在感
3-5步 [动摇]  - 婚礼仪式中，质疑婚姻选择
6-8步 [深入]  - 婚宴间隙，情感突破
9-11步[沦陷]  - 逃离婚礼，背叛选择
12步  [完成]  - 摘下戒指，最终承诺
```

## 契合度计算

- 玩家意图与步骤最佳匹配对比
- 匹配一半以上算高契合
- 高契合度：+10%
- 低契合度：+5%

**完成标准**：
- 80% 以上算完成
- 100% 可触发特殊结局行为

## 记忆连贯性系统

### 等级定义

| 等级 | 条件 | 赵霞反应 |
|------|------|---------|
| 0 | 直接进入场景5 | 完全陌生，像提线木偶 |
| 1 | 完成场景1 | 隐约眼熟，似曾相识 |
| 2 | 完成场景1+2 | 感到命运安排，认出等待的人 |
| 3 | 完成场景1+2+3 | 命中注定，多年感情爆发 |

### 连贯性锁定

- 首次进入场景5时锁定
- 后续进入使用锁定值
- 函数：`lockScene5EntryCoherence(data)`

### 完美记忆路线

- 条件：按顺序完成场景1→2→3后进入场景5
- 效果：记忆混乱度增加被豁免

## 完成度计算

```typescript
export interface Scene5CompletionInfo {
  completionPercent: number;      // 完成百分比
  currentStep: number;            // 当前步骤
  maxAvailableSteps: number;      // 可用最大步骤
  isStepsComplete: boolean;       // 步骤是否完成
  isComplete: boolean;            // 80%以上才算完成
  canTriggerSpecialEnding: boolean; // 100%才能触发
  entryCount: number;             // 进入次数
}
```

## 核心函数

### `generateScene5StepPrompt(data, intent)`

生成当前步骤的AI引导Prompt。

### `generateScene5EntryReplacement(data)`

生成场景5入口替换内容，包含：
- 记忆连贯性提示
- 前置场景摘要
- 部位开发影响

### `calculateScene5Completion(data)`

计算场景5的完成度信息。

### `getRemainingSteps(data)`

计算当前可用的剩余步骤数（19:00触发退出）。

## 自由发挥阶段

12步完成后进入自由互动时间：
- 允许：对话、情感交流、轻度亲密
- 禁止：性行为（触发混乱结局）

时间提示动态调整：
- 剩余>2小时：不提醒
- 剩余1-2小时：轻微暗示
- 剩余<1小时：自然暗示即将结束

## 与其他系统的交互

1. **混乱结局系统**：性行为/打断婚礼触发混乱
2. **记忆连贯性**：影响赵霞初始反应
3. **真好结局**：完成度影响结局质量
4. **部位开发系统**：场景5只能开发精神部位
