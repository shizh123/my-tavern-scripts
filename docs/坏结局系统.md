# 坏结局系统

> 创建日期：2026-01-17
> 文件位置：`src/赵霞/脚本/游戏逻辑/badEndingSystem.ts`
> 状态：已实现

## 概述

坏结局系统处理游戏中的失败结局，主要包括"发现结局"（丈夫怀疑度达到100）和"精神崩溃结局"（记忆混乱度达到100）。

---

## 触发条件

| 结局类型 | 触发条件 |
|---------|---------|
| 发现结局 | 丈夫怀疑度 ≥ 100 |
| 精神崩溃 | 记忆混乱度 ≥ 100 |

---

## 梦境豁免规则

**发现结局在梦境中豁免**：
- 原因：赵霞在睡觉做梦，苏文不可能发现她在梦里做什么
- 检查：`data.世界.游戏阶段 === '梦境'`

**精神崩溃结局不豁免**：
- 原因：这是赵霞自己的精神状态，与是否在梦境无关

---

## 发现结局模板

### 初始触发模板（5种）

**模板1：当场撞见**
```
门突然被推开。
苏文站在门口，手里的公文包"啪"地一声落在地上。
他的眼睛死死地盯着房间里的场景，脸色从惊愕变成铁青。
"你们...在干什么？"
【BAD END: 发现】
```

**模板2：通过监控发现**
```
苏文的声音从身后传来，冰冷得没有一丝温度。
"我早就知道了。"
他举起手机，屏幕上是家里监控录像的画面。
"你以为我什么都不知道？"
【BAD END: 发现】
```

**模板3：邻居告密**
```
苏文回到家时，表情异常平静。
"今天碰到隔壁王阿姨了。"
"说你最近经常有...客人来。说那个人，每次都待很久。"
离婚协议书放在桌上。
【BAD END: 发现】
```

**模板4：突然回家**
```
"我回来拿个文件——"
苏文的话在看到眼前场景的瞬间戛然而止。
"呵。原来如此。原来如此。"
他转身离开，背影决绝。
【BAD END: 发现】
```

**模板5：手机短信暴露**
```
苏文把手机摔在赵霞面前。
屏幕上是一条短信通知，附带着几张照片。
是她和{{user}}的照片。不知道是谁偷拍的。
"我到底做错了什么？"
【BAD END: 发现】
```

### 锁定模板（5种）

触发后持续显示，无论玩家输入什么：

1. **结局已定** - 强调无法改变
2. **无法挽回** - 描写赵霞蜷缩哭泣
3. **空荡的房间** - 苏文离开后的场景
4. **沉默的审判** - 三人僵持的画面
5. **最后的画面** - 苏文离开时的背影

```
[坏结局 - 精神崩溃]

赵霞的精神彻底崩溃了。

梦境与现实的界限变得模糊，她再也分不清什么是真实，什么是虚幻。

"我是谁...？"

她的眼神变得空洞，再也没有了往日的神采。

【BAD END: 精神崩溃】
```

## 核心函数

### `checkBadEnding(data)`

检测是否触发坏结局。

**返回值**：
```typescript
interface BadEndingCheckResult {
  triggered: boolean;           // 是否触发
  type: BadEndingType;          // '发现' | '精神崩溃' | '未触发'
  replaceUserMessage: string | null; // 替换消息
}
```

**检测逻辑**：
1. 检查是否已在坏结局锁定状态（优先级最高）
2. 检查梦境豁免（发现结局在梦境中豁免）
3. 检测发现结局（怀疑度 ≥ 100）
4. 检测精神崩溃（混乱度 ≥ 100）

### `applyBadEndingState(data, type)`

应用坏结局状态到游戏数据：
```typescript
data.结局数据.当前结局 = '坏结局';
data.世界.循环状态 = '结局判定';
```

### `isInBadEndingLock(data)`

检测是否处于坏结局锁定状态：
```typescript
return (
  data.结局数据.当前结局 === '坏结局' &&
  data.世界.循环状态 === '结局判定'
);
```

## ROLL支持

坏结局系统支持ROLL消息检测：
- 即使ROLL到怀疑度未达100的消息
- 只要之前已触发过坏结局
- 系统会使用锁定模板继续显示坏结局

## 类型定义

```typescript
export type BadEndingType = '发现' | '精神崩溃' | '未触发';

export interface BadEndingCheckResult {
  triggered: boolean;
  type: BadEndingType;
  replaceUserMessage: string | null;
}
```

## 与其他系统的交互

1. **境界打断系统**：超阶段行为增加怀疑度
2. **丈夫怀疑度系统**：怀疑度达100触发发现结局
3. **混乱结局系统**：混乱结局优先级高于发现结局
4. **时间系统**：梦境阶段豁免发现结局
