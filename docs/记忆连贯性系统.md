# 记忆连贯性系统

> 创建日期：2026-01-17
> 文件位置：`src/赵霞/脚本/游戏逻辑/dreamKeywordDetection.ts` + `scene5System.ts`
> 状态：已实现

## 概述

记忆连贯性系统决定了赵霞在场景5中对玩家的认知程度。只有按顺序完成场景1→2→3，才能获得最高的记忆连贯性。

## 核心规则

### 连贯性等级

| 等级 | 条件 | 赵霞的反应 |
|------|------|-----------|
| 0 | 直接进入场景5 | 完全陌生，像提线木偶一样做出选择 |
| 1 | 完成场景1 | 隐约觉得眼熟，有似曾相识的感觉 |
| 2 | 完成场景1+2 | 感到命运的安排，认出"曾经等待过的人" |
| 3 | 完成场景1+2+3 | 命中注定的重逢，积压多年的感情爆发 |

### 重要说明
- **场景4时间线在场景5之后**（27岁 vs 25岁），不计入连贯性
- 记忆连贯性在**首次进入场景5时锁定**，后续不会改变

## 数据结构

```typescript
梦境数据: {
  // 动态计算值
  记忆连贯性: number;      // 0-3

  // 进入场景5时的快照（锁定值）
  场景5进入时连贯性?: number;
}
```

## 核心函数

### `calculateMemoryContinuity(data)`

计算当前记忆连贯性等级。

**逻辑**：
```typescript
const 完成场景 = data.梦境数据.已完成场景;
let 连贯性 = 0;

if (完成场景.includes(1)) 连贯性++;
if (完成场景.includes(2)) 连贯性++;
if (完成场景.includes(3)) 连贯性++;

return 连贯性;
```

---

### `lockScene5EntryCoherence(data)`

锁定进入场景5时的记忆连贯性。

**调用时机**：场景5首次进入时

**行为**：
1. 计算当前连贯性
2. 保存到 `场景5进入时连贯性`
3. 后续不再改变

---

### `getScene5LockedCoherence(data)`

获取锁定的场景5进入时连贯性。

**返回值**：锁定值，或未锁定时返回动态计算值

---

### `generateMemoryContinuityPrompt(data, sceneNum)`

生成记忆连贯性提示词。

**场景5专属**：根据连贯性等级生成不同的 AI 提示

## 记忆隔离

### 问题描述
梦境场景之间的服装/外貌被错误继承。

### 解决方案
在提示词中添加【记忆隔离】区块：

```
【⚠️ 记忆隔离 - 非常重要】
记忆连续性只继承**心理记忆**，不继承**物理状态**：
✅ 应该继承：对玩家的感情、认识程度、情感变化、互动经历
❌ 不应继承：服装、外貌、妆容、年龄外表、场景环境

当前场景是XX岁的赵霞：
- 她的外貌、穿着必须符合XX岁的设定
- 完全忽略对话历史中其他场景的服装描写
- 每个梦境场景的服装由AI根据当前场景年龄自由创作
```

- 记忆连贯性 = 3

### 特殊待遇
