# 状态栏系统

> 创建日期：2026-01-17
> 文件位置：`src/赵霞/界面/状态栏/`
> 状态：已实现

## 概述

状态栏系统负责在游戏界面中显示当前时间、境界、数值等游戏状态信息。根据"已进入过梦境"自动切换显示文本（纯爱模式 vs 真相模式），真相模式还会额外显示威胁数值和苦主视角。

---

## 核心功能

1. **显示当前时间**：Day X, HH:00 格式
2. **显示境界**：根据模式显示不同名称
3. **显示核心数值**：依存度、道德底线
4. **显示部位进度**：5个部位的开发进度
5. **威胁数值**（仅真相模式）：记忆混乱度、丈夫怀疑度
6. **苦主视角**（仅真相模式）：显示丈夫的心理活动

## 核心函数

### `renderProgressBar(value, max, width)`

生成进度条字符串。

```typescript
function renderProgressBar(value: number, max: number = 100, width: number = 20): string {
  const percentage = Math.min(100, Math.max(0, (value / max) * 100));
  const filled = Math.floor((percentage / 100) * width);
  const empty = width - filled;
  return `[${'█'.repeat(filled)}${'░'.repeat(empty)}] ${value}/${max}`;
}
```

### `renderMainStatus(data)`

渲染主状态栏，包含时间、境界、数值、部位进度、威胁状态。

### `renderHusbandPerspective(data)`

渲染苦主视角区块，仅在满足以下条件时显示：
- 真相模式（已进入过梦境）
- 境界 ≥ 2
- 日常阶段
- 苏文在家

### `renderFullStatus(data)`

渲染完整状态栏（主状态 + 苦主视角）。

## 苦主视角条件

使用统一的条件判断函数 `shouldGenerateHusbandThought`：

```typescript
function renderHusbandPerspective(data: SchemaType): string {
  if (!shouldGenerateHusbandThought(data)) {
    return '';
  }
  // ...渲染逻辑
}
```

## 数据刷新机制

状态栏通过以下事件触发刷新：

1. **MESSAGE_RECEIVED**：新消息接收时
2. **MESSAGE_SWIPED**：ROLL 操作时
3. **GENERATION_ENDED**：AI 生成结束时
4. **手动刷新**：`状态栏需要刷新` 标记为 true 时
