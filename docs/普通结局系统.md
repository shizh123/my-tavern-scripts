# 普通结局系统

> 创建日期：2026-01-17
> 文件位置：`src/赵霞/脚本/游戏逻辑/normalEndingSystem.ts`
> 状态：已实现

## 概述

普通结局是玩家未能打破时间循环时触发的结局。时间循环重置，一切回到原点。

---

## 触发条件

| 条件 | 要求 |
|------|------|
| 时间 | Day 5, 20:00+ |
| 坏结局 | 未触发（怀疑度<100 且 混乱度<100） |
| 场景完成 | 未完成全部5个梦境场景 |

---

## 触发逻辑

```typescript
export function shouldTriggerNormalEnding(data: SchemaType): boolean {
  // 检查坏结局条件
  const isBadEnding =
    data.现实数据.丈夫怀疑度 >= 100 ||
    data.梦境数据.记忆混乱度 >= 100;
  if (isBadEnding) return false; // 应触发坏结局

  // 检查好结局条件
  const completedScenes = new Set(data.梦境数据.已完成场景);
  const allScenesComplete =
    completedScenes.size === 5 &&
    [1, 2, 3, 4, 5].every(s => completedScenes.has(s));
  if (allScenesComplete) return false; // 应触发好结局

  // 未满足好结局，也未触发坏结局 → 普通结局
  return true;
}
```

## 获取第一条AI消息

使用酒馆助手API获取开局问候：

```typescript
function getFirstAIMessage(): string | null {
  try {
    const messages = getChatMessages(0);
    if (messages && messages.length > 0) {
      const firstMessage = messages[0];
      if (firstMessage.role === 'assistant' && firstMessage.message) {
        return firstMessage.message;
      }
    }
    return null;
  } catch (err) {
    console.error('[普通结局系统] 获取第一条AI消息失败:', err);
    return null;
  }
}
```

## 核心函数

### `checkNormalEnding(data)`

检测普通结局。

**返回值**：
```typescript
interface NormalEndingCheckResult {
  triggered: boolean;
  type: NormalEndingType;
  replaceUserMessage: string | null;
  firstGreeting: string | null;
}
```

### `applyNormalEndingState(data)`

应用普通结局状态：
```typescript
data.结局数据.当前结局 = '普通结局';
data.世界.循环状态 = '结局判定';
```

### `isInNormalEndingLock(data)`

检测是否处于普通结局锁定状态。

## 设计理念

- 玩家没有发现梦境入口
- 或没有完成足够的场景
- 时间循环重置，一切回到原点
- 玩家不保留记忆，只看到循环重置的结果

---

## 与其他系统的交互

1. **结局判定系统**：在checkEnding()中调用
2. **坏结局系统**：坏结局优先级高于普通结局
3. **好结局系统**：好结局优先级高于普通结局
