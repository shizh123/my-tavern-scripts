# 双轨系统

> 创建日期：2026-01-17
> 文件位置：`src/赵霞/脚本/游戏逻辑/dualTrackSystem.ts`
> 状态：已实现

## 概述

双轨系统实现了"苦主视角"功能，在真相模式下，AI 会在回复中额外生成苏文的心理活动，通过 `<HusbandThought>` 标签包裹。

## 核心概念

- **苦主**：苏文（丈夫），被 NTR 的对象
- **苦主视角**：以苏文的第一人称视角描述他的心理活动
- **显示条件**：真相模式 + 境界≥2 + 苏文在家 + 日常阶段

---

## 类型定义

本系统涉及两种不同层面的类型定义：

### 1. 路线类型（RouteType）- 游戏进度阶段

用于判断玩家当前所处的游戏进度阶段，影响 AI 描写风格和苦主视角生成。

```typescript
type RouteType = '序章' | '纯爱日常' | '真相日常' | '真相梦境' | '结局';
```

| 条件 | 路线类型 | 说明 |
|------|---------|------|
| 游戏阶段='序章' | 序章 | 游戏开始，尚未进入正式流程 |
| 游戏阶段='结局' | 结局 | 结局判定阶段 |
| 未进入过梦境 | 纯爱日常 | 伪装成纯爱游戏的日常 |
| 已进入过梦境 + 游戏阶段='日常' | 真相日常 | 揭示真相后的日常阶段 |
| 已进入过梦境 + 游戏阶段='梦境' | 真相梦境 | 揭示真相后的梦境阶段 |

### 2. 场景类型 - 当前场景位置

用于判断玩家当前是在现实还是梦境中，影响系统行为（如打断系统、惩罚转换等）。

```typescript
type SceneType = '现实' | '梦境';
```

| 场景类型 | 判断条件 | 系统行为差异 |
|---------|---------|-------------|
| 现实 | `游戏阶段 !== '梦境'` | 正常触发打断系统、怀疑度惩罚 |
| 梦境 | `游戏阶段 === '梦境'` | 豁免打断系统、怀疑度转为混乱度 |

### 两者的关系

- **路线类型**：宏观层面，描述玩家的游戏进度和体验模式
- **场景类型**：微观层面，描述玩家当前所处的空间位置

例如：一个处于"真相梦境"路线的玩家，其场景类型是"梦境"。

## 苦主视角生成条件

```typescript
function shouldGenerateHusbandThought(data): boolean {
  // 1. 必须是真相模式
  if (!data.世界.已进入过梦境) return false;

  // 2. 必须是日常阶段（梦境中不输出）
  if (data.世界.游戏阶段 !== '日常') return false;

  // 3. 境界必须≥2
  if (data.赵霞状态.当前境界 < 2) return false;

  // 4. 苏文必须在家
  if (data.现实数据.丈夫当前位置 === '外出') return false;

  return true;
}
```

### `getCurrentRouteType(data)`

**返回值**：`'序章' | '纯爱日常' | '真相日常' | '真相梦境' | '结局'`

---

### `shouldGenerateHusbandThought(data)`

判断是否应该生成苦主视角。

**返回值**：`boolean`

---

### `parseHusbandThought(aiText)`

从 AI 回复中解析苦主视角内容。

**匹配模式**：`<HusbandThought>...</HusbandThought>`

**返回值**：解析出的心理活动文本，或 `null`

## 标签处理

AI 回复中可能包含的标签：

```
<HusbandThought>
苏文的内心活动...她今天怎么穿得这么暴露...
</HusbandThought>
```

### 处理流程

1. **解析**：`parseHusbandThought()` 提取内容
2. **移除**：从 AI 回复中删除标签，避免显示在正文中
3. **存储**：保存到 `data.现实数据.丈夫心理活动`
4. **广播**：发送 `HUSBAND_PERSPECTIVE` 事件到前端

### 不当生成处理

如果 AI 在不应该生成的场景（如梦境）生成了苦主视角：
- 仍然移除标签
- 记录警告日志
- **不存储**到数据中

## 苦主视角内容指导

### 境界2（迷途）
- 困惑、不安
- 察觉到妻子的变化但无法确定
- 自我安慰

### 境界3（溺深）
- 明显的危机感
- 想要挽回但不知道如何
- 内心挣扎

### 境界4（归虚）
- 绝望、无力
- 开始接受现实
- 可能的愤怒或自我否定

### 境界5（焚誓）
- 完全臣服或完全崩溃
- 绿帽癖萌芽（真爱结局路线）
- 最终的接受

## 与其他系统的交互

1. **游戏逻辑主入口**：解析和清理 AI 回复中的标签
2. **状态栏**：显示苏文心理活动
3. **结局系统**：真爱结局中苏文的心理转变
