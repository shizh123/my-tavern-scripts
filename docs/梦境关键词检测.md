# 梦境关键词检测系统

> 创建日期：2026-01-17
> 文件位置：`src/赵霞/脚本/游戏逻辑/dreamKeywordDetection.ts`
> 状态：已实现

## 概述

梦境关键词检测系统负责在梦境阶段检测玩家输入和AI回复中的关键词，实现部位开发进度的自动更新、场景正确答案判定、以及梦境摘要生成。

---

## 核心功能

1. **部位开发检测**：检测玩家输入中的部位关键词，更新开发进度
2. **场景正确答案判定**：判断玩家在每个场景中的选择是否正确
3. **记忆连贯性计算**：计算玩家完成场景的顺序得分
4. **梦境摘要生成**：生成玩家行为记录摘要

## 部位进度检测

### `detectBodyPartProgress(text)`

检测文本中的部位关键词，返回各部位的进度增量：

```typescript
export function detectBodyPartProgress(text: string): Record<BodyPartName, number> {
  const result = { 嘴巴: 0, 胸部: 0, 下体: 0, 后穴: 0, 精神: 0 };

  for (const [part, keywords] of Object.entries(BODY_PART_KEYWORDS)) {
    const matchCount = keywords.filter(kw => text.includes(kw)).length;
    if (matchCount > 0) {
      result[part] = Math.min(10, matchCount * 5); // 每个关键词+5，最多+10
    }
  }

  return result;
}
```

### `updateBodyPartProgress(data, progress)`

更新部位进度到游戏数据：

```typescript
export function updateBodyPartProgress(
  data: SchemaType,
  progress: Record<BodyPartName, number>
): void {
  for (const [part, value] of Object.entries(progress)) {
    if (value > 0) {
      const oldValue = data.赵霞状态.部位进度[part];
      const newValue = Math.min(100, oldValue + value);
      data.赵霞状态.部位进度[part] = newValue;
    }
  }
}
```

```typescript
export const SCENE_CORRECT_ANSWERS: Record<number, BodyPartName[]> = {

  2: ['胸部'],           // 场景2：等待（17岁）- 胸部觉醒
  3: ['下体'],           // 场景3：生日（23岁）- 初夜
  4: ['后穴'],           // 场景4：争吵（28岁）- 后穴开发
  5: ['精神'],           // 场景5：婚礼（25岁）- 精神改写
};
```

### `checkSceneCorrectAnswer(data, sceneNum)`

判断场景答案是否正确：

```typescript
export function checkSceneCorrectAnswer(data: SchemaType, sceneNum: number): boolean {
  const 正确部位 = SCENE_CORRECT_ANSWERS[sceneNum] || [];
  const 选择部位 = getSceneSelectedParts(data, sceneNum);

  // 检查是否选择了所有正确部位
  return 正确部位.every(part => 选择部位.includes(part));
}
```

### `processSceneCompletion(data, sceneNum)`

处理场景完成判定：

```typescript
export function processSceneCompletion(data: SchemaType, sceneNum: number): void {
  const isCorrect = checkSceneCorrectAnswer(data, sceneNum);

  // 标记场景完成
  if (!data.梦境数据.已完成场景.includes(sceneNum)) {
    data.梦境数据.已完成场景.push(sceneNum);
  }

  // 如果正确，添加到正确重构列表
  if (isCorrect && !data.梦境数据.正确重构场景.includes(sceneNum)) {
    data.梦境数据.正确重构场景.push(sceneNum);
  }
}
```

## 记忆连贯性系统

### 计算规则

记忆连贯性基于场景完成顺序：

| 等级 | 条件 | 赵霞的反应 |
|------|------|------------|
| 0 | 直接进入场景5 | 完全陌生 |
| 1 | 完成场景1 | 隐约眼熟 |
| 2 | 完成场景1+2 | 认出"等待的人" |
| 3 | 完成场景1+2+3 | 命中注定的重逢 |

**注意**：场景4时间线在场景5之后（27岁 vs 25岁），不计入连贯性。

### `calculateMemoryContinuity(data)`

```typescript
export function calculateMemoryContinuity(data: SchemaType): number {
  const 完成场景 = data.梦境数据.已完成场景;
  let 连贯性 = 0;

  if (完成场景.includes(1)) 连贯性++;
  if (完成场景.includes(2)) 连贯性++;
  if (完成场景.includes(3)) 连贯性++;

  return 连贯性;
}
```

### `generateMemoryContinuityPrompt(data, sceneNum)`

生成记忆连贯性提示词（给AI使用）：

```typescript
export function generateMemoryContinuityPrompt(data: SchemaType, sceneNum: number): string {
  if (sceneNum !== 5) return '';

  const 连贯性 = calculateMemoryContinuity(data);

  switch (连贯性) {
    case 3: return '【完美记忆】赵霞完整记得所有场景...';
    case 2: return '【高连贯性】赵霞记得大部分场景...';
    case 1: return '【低连贯性】赵霞有些模糊的记忆...';
    default: return '【无连贯性】赵霞完全不认识你...';
  }
}
```

## 记忆隔离机制

### 问题

梦境场景之间的服装/外貌被错误继承。

### 解决方案

在提示词中添加记忆隔离规则：

```
【记忆隔离】
记忆连续性只继承心理记忆，不继承物理状态：
✅ 应该继承：对玩家的感情、认识程度、情感变化
❌ 不应继承：服装、外貌、妆容、年龄外表、场景环境
```
