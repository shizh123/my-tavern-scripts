## 概述

境界系统定义了赵霞与玩家关系的5个阶段。两种模式下境界名称和允许的互动行为完全不同，但底层逻辑相同。

## 境界计算

真相模式下，境界由依存度自动计算：

```typescript
export function calculateRealmFromDependency(依存度: number): number {
  if (依存度 >= 80) return 5;  // 焚誓
  if (依存度 >= 60) return 4;  // 归虚
  if (依存度 >= 40) return 3;  // 溺深
  if (依存度 >= 20) return 2;  // 迷途
  return 1;                     // 初染
}
```

### 依存度 → 境界对应表

| 依存度范围 | 境界 |
|------------|------|
| 0-19 | 1 |
| 20-39 | 2 |
| 40-59 | 3 |
| 60-79 | 4 |
| 80-100 | 5 |

## 部位解锁规则

境界决定了哪些部位可以互动：

```typescript
export const REALM_BODY_PART_UNLOCK: Record<number, string[]> = {
  1: ['嘴巴'],                              // 轻吻（有程度限制）
  2: ['嘴巴', '胸部'],                      // + 胸部
  3: ['嘴巴', '胸部', '下体'],              // + 下体
  4: ['嘴巴', '胸部', '下体', '后穴'],      // + 后穴
  5: ['嘴巴', '胸部', '下体', '后穴', '精神'], // 全部解锁
};
```

### 程度限制

部位解锁不代表可以进行任何程度的互动，每个境界有对应的**程度上限**：

| 境界 | 嘴巴程度 | 说明 |
|------|----------|------|
| 1 | 轻吻 | 仅允许轻度接触，**禁止**舌吻、深吻等色情行为 |
| 2 | 普通亲吻 | 可以更亲密，但仍有分寸 |
| 3+ | 无限制 | 所有程度均可 |

**注意**：真相模式下没有境界限制，所有部位可同时开发。境界只影响服装妆容和AI描写风格。

## 核心函数

### `getRealmConstraints(realm, 已进入过梦境)`

获取当前境界的外观约束。

### `getStyleGuidance(realm, 已进入过梦境)`

获取当前境界的风格指导（给AI的建议）。

### `getRealmTitle(realm, 已进入过梦境)`

获取当前境界的标题/名称。

### `getAllowedInteractions(realm, 已进入过梦境)`

获取当前境界允许的互动列表。

### `isActionAllowed(行为类型, 当前境界, 已进入过梦境)`

检查行为是否被允许。

## 与其他系统的交互

1. **依存度系统**：依存度决定境界
2. **外观系统**：境界决定服装妆容约束
3. **打断系统**：境界决定允许的互动范围
4. **状态栏系统**：显示当前境界名称
5. **文本映射系统**：境界名称的纯爱/真相模式映射
