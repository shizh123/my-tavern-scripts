# 完美真爱结局系统

> 创建日期：2026-01-17
> 文件位置：`src/赵霞/脚本/游戏逻辑/perfectTrueEndingSystem.ts`
> 状态：已实现

## 概述

完美真爱结局是游戏的"真正最佳结局"，需要满足最高条件才能触发。这是一个带有仪式感的结局，赵霞和苏文共同为玩家准备了一场特殊的"婚礼"。

**核心主题**：不是堕落，而是"真爱"——赵霞真心选择了玩家，苏文也选择了成全。

---

## 与真好结局的区别

完美真爱结局与真好结局是**完全独立的两套系统**：

| 对比项 | 完美真爱结局 | 真好结局 |
|--------|--------------|----------|
| **结局标题** | 【PERFECT TRUE END】命中注定 | 【TRUE END】禁忌之爱 |
| **触发条件** | 5场景全完成+全正确+记忆连贯性=3 | 5场景全完成+全正确+记忆连贯性<3 |
| **阶段数量** | 12阶段（婚礼仪式） | 10阶段（堕落剧情） |
| **苏文状态** | 主动成全，成为"苦主" | 被药倒，昏迷不醒 |
| **赵霞状态** | 纯白婚纱，发自内心 | 情趣内衣，被改造 |
| **氛围基调** | 真爱的选择 | 堕落的胜利 |
| **身体展示** | 无纹身、无乳环等改造 | 展示淫纹、乳环等改造 |
| **代码文件** | `perfectTrueEndingSystem.ts` | `trueEndingSystem.ts` |

## 触发条件

| 条件 | 要求 | 说明 |
|------|------|------|
| 时间 | Day 5, 11:00+ | 从场景5醒来后1小时开始引导 |
| 循环状态 | 结局判定 或 进行中 或 已破解 | - |
| 场景完成 | 5个场景全部完成 | 场景1-5 |
| 正确场景数 | = 5（全部正确） | 所有场景都选择正确 |
| **记忆连贯性** | **= 3** | **完美记忆路线** |

### 完美记忆路线

玩家需要在**进入场景5之前**，按顺序完成场景1→2→3，才能达到记忆连贯性=3。

这意味着赵霞拥有完整的记忆重构：
- 16岁那个夏天的初恋
- 17岁在路口的等待
- 20岁生日想要逃离的渴望
- 结婚那天心中真正的人

## 12阶段婚礼仪式详细设计

### 阶段0：序幕 - 命中注定的早晨

- **时间**：11:00
- **场景**：从梦境醒来，赵霞终于明白了自己的感情
- **锚点**：记忆觉醒、心意已决
- **最少轮数**：2轮
- **赵霞状态**：不是迷茫，而是清醒；不是被改造，而是终于想起

### 阶段1：长谈 - 与苏文的坦白

- **场景**：赵霞决定和苏文坦诚一切
- **关键**：这是一场漫长而艰难的对话
- **锚点**：开始谈话、坦白感情、苏文反应
- **最少轮数**：3轮
- **苏文反应**：震惊 → 愤怒 → 痛苦 → 沉默 → 释然

### 阶段2：决定 - 苏文的成全

- **场景**：苏文做出艰难的决定
- **关键**：他选择成全他们
- **锚点**：苏文决定、成全宣言
- **最少轮数**：2轮
- **核心台词**："我没有办法强迫一个人爱我。"

### 阶段3：准备 - 婚礼的筹备

- **场景**：苏文默默准备一切，赵霞和玩家等待
- **关键**：漫长的下午，等待那个时刻
- **锚点**：等待时光、婚礼准备
- **最少轮数**：2轮

### 阶段4：蒙眼 - 蒙眼带入

- **时间**：20:00，婚礼仪式正式开始
- **场景**：苏文蒙住玩家的眼睛，带他走向准备好的房间
- **锚点**：蒙眼、带入房间
- **最少轮数**：2轮

### 阶段5：揭幕 - 揭开眼罩

- **场景**：眼罩被取下，眼前是一个小型的婚礼现场
- **布置**：白色纱幔、鲜花、蜡烛、红毯、摄像机
- **锚点**：揭开眼罩、看到场景
- **最少轮数**：2轮

### 阶段6：新娘 - 看到赵霞

- **场景**：赵霞穿着纯白的婚纱，站在房间中央
- **装扮**：纯白婚纱、头纱、白色百合，**无任何淫秽装饰**
- **锚点**：看到新娘、婚纱描写
- **最少轮数**：2轮

### 阶段7：司仪 - 苏文的宣言

- **场景**：苏文作为司仪，宣布仪式开始，并说出他的决定
- **关键**：讲述昨晚的长谈，宣布成全
- **锚点**：仪式开始、苏文讲述、苏文成全承诺
- **最少轮数**：3轮
- **核心承诺**：搬到原来的房间、继续支持这个家、抚养他们的孩子

### 阶段8：告白 - 赵霞的誓言

- **场景**：赵霞走向玩家，说出她压抑多年的心声
- **内容**：回忆16岁、17岁、20岁的每一刻
- **锚点**：走向玩家、回忆往事、告白誓言
- **最少轮数**：3轮
- **核心台词**："我爱你。从今以后，我只属于你。"

### 阶段9：誓言 - 玩家的誓言

- **场景**：玩家向赵霞说出自己的誓言
- **关键**：这是玩家的高光时刻
- **锚点**：玩家誓言、回应告白
- **最少轮数**：2轮

### 阶段10：戒指 - 交换戒指

- **场景**：赵霞摘下原来的戒指，换上属于他们的新戒指
- **仪式**：摘下旧戒指 → 递给苏文 → 戴上新戒指
- **锚点**：摘下旧戒指、戴上新戒指
- **最少轮数**：2轮

### 阶段11：完成 - 新人之吻

- **场景**：苏文宣布仪式完成，新人接吻
- **结果**：完美真爱结局达成
- **锚点**：宣布完成、新人之吻、结局达成
- **最少轮数**：1轮

## 时间线设计

> **设计说明**：完美真爱结局已突破时间循环，因此时间要求不严格，可以延续到Day 6甚至更长。

```
Day 5:
├── 10:00        场景5第2次退出
├── 11:00        开始结局引导（阶段0：序幕）
├── 11:00-19:00  阶段0-3（序幕 → 长谈 → 决定 → 准备）
├── 20:00        婚礼仪式开始（阶段4：蒙眼）
├── 21:00        【玩家自由时间 1小时】
├── 22:00        仪式继续（阶段5-7：揭幕 → 新娘 → 司仪）
├── 23:00        【玩家自由时间 1小时】
└── 00:00        仪式高潮（阶段8-11：告白 → 誓言 → 戒指 → 完成）

Day 6+:（如果Day 5未完成）
└── 继续婚礼仪式，无时间限制
```

## 苏文的角色定位

在完美真爱结局中，苏文是一个**悲情英雄**：

### 心理转变
| 阶段 | 苏文的状态 |
|------|-----------|
| 长谈前 | 察觉异常但不愿面对 |
| 长谈中 | 震惊 → 愤怒 → 痛苦 |
| 长谈后 | 接受现实，做出成全的决定 |
| 仪式中 | 平静地担任司仪 |
| 结局后 | 接受新角色，默默守护 |

### 核心动机
- 他真的爱赵霞
- 他无法强迫一个人爱他
- 如果让她幸福的不是他，他至少可以目送她走向幸福
- 成全是他表达爱的方式

### 承诺内容
- 搬到原来的房间去住
- 经济上继续支持这个家
- 抚养他们将来的孩子
- 保密，维持表面的家庭形象

## AI描写指南

### 推荐元素
- ✅ 纯白、优雅的婚纱
- ✅ 温馨的婚礼布置（纱幔、鲜花、蜡烛）
- ✅ 真挚的眼神和告白
- ✅ 仪式感的动作（交换戒指、接吻）
- ✅ 感动、幸福的氛围
- ✅ 强调"爱"、"选择"、"永远"
- ✅ 赵霞的泪水是幸福的泪水
- ✅ "命中注定"、"终于等到你"的浪漫感

### 禁止元素
- ❌ 纹身
- ❌ 穿环（乳环、阴蒂环等）
- ❌ 淫秽的身体装饰
- ❌ 过于色情的婚纱
- ❌ 粗俗的语言
- ❌ 强调"堕落"、"淫荡"等词
- ❌ 让赵霞表现出迷茫或被控制感
- ❌ 羞辱苏文
