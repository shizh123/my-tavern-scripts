# 场景5多次进入机制

> 创建日期：2026-01-17
> 状态：设计完成

## 概述

场景5（花嫁的誓约）采用12步线性剧情系统，需要玩家多次进入才能完成全部步骤。

**核心主题**：婚礼的改写 - 玩家在赵霞的结婚日记忆中出现，让她对婚姻产生动摇，最终选择离开苏文。

**场景背景**：
- 时间设定：赵霞25岁，与苏文结婚当天
- 场景环境：婚礼现场/酒店
- 玩家身份：以"陌生人"身份出现，但赵霞隐约觉得很熟悉（取决于记忆连贯性）

---

## 12步剧情设计

| 阶段 | 步骤 | 标题 | 描述 |
| ---- | ---- | ---- | ---- |
| 初入 | 1 | 婚礼前的化妆间 | 赵霞穿着婚纱，独自坐在化妆镜前，玩家以"陌生人"身份出现 |
| 初入 | 2 | 第一次对话 | 赵霞注意到玩家，感到困惑但不抗拒，隐约觉得眼前人很熟悉 |
| 动摇 | 3 | 婚礼仪式前 | 宾客入场，苏文在另一边准备，玩家开始质疑她的选择 |
| 动摇 | 4 | 仪式途中 | 赵霞走向红毯，但目光不自觉看向玩家 |

## 契合度系统

每一步会分析玩家输入的意图（态度/行为/目标），与最佳匹配对比：

| 维度 | 可能值 | 说明 |
| ---- | ------ | ---- |
| 态度 | 温柔/激进/被动 | 玩家行动的方式 |
| 行为 | 言语/肢体/心理 | 玩家行动的类型 |
| 目标 | 安慰/质疑/引诱 | 玩家行动的目的 |

**进度累加规则**：
- 高契合度（匹配一半以上）：+10%
- 低契合度：+5%
- 80%以上算完成，100%可触发特殊结局行为

---

## 记忆连贯性系统

进入场景5时自动计算，基于是否按顺序完成场景1-2-3：

| 等级 | 条件 | 赵霞的反应 |
| ---- | ---- | ---------- |
| 0级 | 直接进入场景5 | 完全陌生，像提线木偶一样做出选择 |
| 1级 | 完成场景1 | 隐约觉得眼熟，有似曾相识的感觉 |
| 2级 | 完成场景1+2 | 感到命运的安排，认出"曾经等待过的人" |
| 3级 | 完成场景1+2+3 | 命中注定的重逢，积压多年的感情爆发 |

**注意**：场景4时间线在场景5之后（27岁 vs 25岁），不计入连贯性。

## 第一次进入

### 时间
- 进入时间：08:00-19:00（使用安眠药）
- 退出时间：19:00触发退出描写，20:00游戏阶段切换为日常

### 完成内容
- 步骤1-2（初入）：玩家在化妆间出现
- 步骤3-5（动摇）：婚礼仪式中，赵霞开始动摇
- 步骤6-8（深入）：婚宴间隙，情感突破
- 步骤9-10（沦陷）：赵霞想要逃离婚礼

### 退出状态
- 步骤进度保存为10
- 精神进度根据完成情况累加

## 第二次进入

### 时间
- 进入时间：08:00-19:00（使用安眠药）
- 退出时间：19:00触发退出描写，20:00游戏阶段切换为日常

### 完成内容
1. **步骤11-12**：仪式结尾
   - 步骤11：赵霞想要逃离
   - 步骤12：赵霞摘下结婚戒指，做出最终承诺

2. **自由互动阶段**：步骤12完成后 → 19:00退出

## 自由互动阶段

### 触发时机
步骤12完成后，直到19:00触发退出描写

### 场景状态
- 婚礼仪式刚结束
- 赵霞穿着婚纱
- 两人独处

### 允许行为
| 类型 | 示例 |
|------|------|
| 对话 | 聊天、回忆过去、展望未来 |
| 情感交流 | 告白、表达爱意 |
| 轻度亲密 | 拥抱、亲吻、牵手、依偎 |

### 禁止行为
| 类型 | 示例 |
|------|------|
| 性行为 | 插入、口交、乳交等涉及性器官的行为 |

### AI描写要点
- 描写婚礼后的温馨氛围，赵霞的幸福和满足
- 两人可以回忆过去的点点滴滴（场景1-4的记忆）
- 赵霞会撒娇、依偎、表达爱意

### 动态时间提示（2026-01-17 更新）

根据剩余时间动态决定是否暗示"梦要醒了"：

| 剩余时间 | AI行为 |
|---------|--------|
| >2小时 | 完全不提退出，专注温馨互动 |
| 1-2小时 | 不主动提及，专注当前互动 |
| <1小时 | 可以自然暗示即将结束（场景边缘模糊、光线变柔和等） |

**不要每句话都提"快醒了"**，只在最后时刻自然带入。

## 混乱结局触发

### 触发条件
在场景5中尝试性行为

### 容错次数

| 前置条件 | 性行为次数限制 | 说明 |
|---------|---------------|------|
| 未完成场景3 | 2次触发混乱 | 第1次警告，第2次混乱结局 |
| 已完成场景3 | 3次触发混乱 | 有记忆基础，多1次容错 |
| 打断婚礼仪式（步骤3-5） | 直接触发 | 无容错 |

### 设计理念
- 这是结婚日记忆，性行为会严重扭曲记忆结构
- 正常玩家按剧情走不会触发
- 作死玩家在自由互动阶段尝试性行为 → 触发混乱结局

---

## 与结局的关系

### 真爱结局条件
- 完成全部5个梦境场景（包括场景5的12步）
- 所有场景选择正确（正确重构场景 = 5）
- 记忆连贯性 = 3（按顺序完成场景1→2→3后进入场景5）

### 场景5对结局的影响
- 完成12步 = 场景5完成 = 正确重构场景+1
- 精神进度达到要求
- 不触发混乱结局
