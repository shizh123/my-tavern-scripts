## 一、核心概念

玩家同时经营两条调教线：

1. **主线（现实）**：云霜凝治疗 + 苗广绿帽线（已实现）
2. **副线（神魂空间）**：苗喧未婚妻的神魂空间调教

两条线通过苗喧形成叙事交叉——苗喧在现实中向未婚妻倾诉家庭崩坏，而他不知道同一个人（{{user}}）正在她的神魂空间里调教她。父子两代走同一条路。

## 二、苗喧升级：碎片 → 完整面板

### 旧系统（废弃）

苗喧原为纯叙事彩蛋——每10-15楼一段80-150字碎片旁白，无变量、无互动。8个情感池（A-H）跟随苗广心态被动切换。

### 新系统

苗喧拥有独立状态面板，成为有数值、有心态演变的完整角色。

#### 变量结构（草案）

| 变量 | 类型 | 初始值 | 说明 |
|------|------|--------|------|
| 心态 | enum | 蔑视 | 对{{user}}的态度，见下方演变 |
| 对未婚妻信任度 | number 0→100 | 80 | 他对未婚妻的信赖程度 |
| 警觉度 | number 0→100 | 0 | 对家庭异常的认知程度 |
| 心理活动 | string | — | AI每轮生成，跟随心态变化 |

#### 心态演变

| 心态 | 触发条件 | 说明 |
|------|---------|------|
| 蔑视 | 初始 | 看不起{{user}}，认为只是庸医 |
| 困惑 | 苗广进入疑惑/察觉 | 发现父亲行为异常 |
| 不安 | 苗广进入屈辱 | 家庭气氛明显变化，开始向未婚妻倾诉 |
| 恐惧 | 苗广进入默许/沉溺 | 目睹父亲彻底改变，无力阻止 |
| 崩溃 | 千晶幻术完成 | 父亲记忆被改写，全家只有他知道真相 |
| 失去 | 掌门改嫁完成 | 家庭结构彻底重塑，他被边缘化 |

#### 面板位置

- **日常/云霜凝神魂空间**：苗喧面板嵌入苗广面板区域（作为子板块或独立tab）
- **未婚妻神魂空间**：苗喧面板成为主面板之一，显示他对未婚妻的心理状态

## 四、神魂空间切换机制

### 解锁条件

新婚夜记忆场景（云霜凝神魂空间阶段3首次触发）完成后，玩家进入神魂空间时出现选择：

```
⟨ 云霜凝·神魂空间 ⟩    ⟨ ???·神魂空间 ⟩
```

初期不知道她是谁，随互动推进逐步揭示身份。

### 状态栏切换

```
┌─────────────────────────────────────────────┐
│  日常 / 云霜凝神魂空间                         │
│                                              │
│  [云霜凝面板]  [苗广面板+苗喧面板]  [商店]      │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  未婚妻神魂空间                               │
│                                              │
│  [未婚妻面板]  [苗喧面板]  [商店?]             │
└─────────────────────────────────────────────┘
```

进入未婚妻神魂空间时：
- 状态栏整体切换为未婚妻视图
- 云霜凝面板 → 未婚妻面板
- 苗广面板隐藏，苗喧面板保留（但显示内容变化：他对未婚妻的心理，而非对家庭的观察）
- 商店是否共用或独立待定

退出时恢复原状态栏。

## 五、叙事交叉设计

### 信息差

玩家能同时看到两面，苗喧只看到一面：

| 苗喧看到的 | 实际发生的 |
|-----------|-----------|
| "她说最近在闭关修炼" | 她在神魂空间被{{user}}调教 |
| "她最近有点心不在焉" | 她在想神魂空间里的事 |
| "至少她还在我身边" | 她内心已经倒向{{user}} |
| "这个家只有她还正常" | 她是下一个"云霜凝" |

### 苗喧碎片的进化

苗喧面板中的心理活动应体现两条线的交织：

- **主线相关**："父亲今天又跪在那个人面前了……"
- **副线相关**："她说在修炼……可她从来不在闭关的时候笑。"
- **交叉**："这个家还剩什么是真的？"

            └→ 千晶幻术（阶段≥7, 需寝取宣告已完成）
                    └→ 掌门改嫁（阶段≥8, 需千晶幻术完成）
