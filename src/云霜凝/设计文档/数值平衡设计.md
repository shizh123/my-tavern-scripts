## 三、注入方式说明

脚本向AI注入信息的3种方式：

| 方式 | 机制 | 是否立即触发AI回复 | 用途 |
|------|------|------------------|------|
| **方式1** | 写入 `_待发送道具事件`，等玩家下次发消息时附加到 user 消息 | 否，等玩家主动发消息 | 装备切换等不需要立即AI反馈的操作 |
| **方式2** | `getStatusSnapshot()` → push system message 到 chat 尾部 | 否，每轮自动注入 | 状态快照，告诉AI当前数值/道具/阶段 |
| **方式3** | 写入 `_待发送道具事件` + `triggerSlash('/send （使用道具）|/trigger')` | **是，立即触发** | 消耗品使用、体改仪式、性癖觉醒等需要AI即时生成剧情的场景 |

代码位置：
- 方式1/3写入：`shopSystem.ts` 中 `processNewlyActivatedItems`
- 方式1/3读取：`index.ts` 事件1 `CHAT_COMPLETION_PROMPT_READY`
- 方式2：`index.ts` 事件1 中调用 `getStatusSnapshot()`
- 方式3触发：`ShopPanel.vue` 中 `triggerSlash`

## 四、消耗品冷却系统

消耗品使用后进入楼层冷却，冷却期间无法再次使用同一消耗品。按150楼结局反推，消耗品贡献约40%数值变化。

| 消耗品 | 效果 | 冷却楼层 | 150楼最多用 |
|--------|------|---------|------------|
| 安神香 | 防线-5 | 15楼 | ~10次(降50) |
| 蚀心露 | 防线-3, 信任+3, 疑心+3 | 25楼 | ~6次(降18/加18/疑心+18) |
| 定心符 | 疑心-8 | 20楼 | ~7次 |
| 混沌珠 | 疑心值降至区间下限（仅屈辱前） | 40楼 | ~3次 |
| 神魂共鸣石 | 信任+5 | 12楼 | ~12次(加60) |

需记录每个消耗品上次使用的楼层号，用于冷却判断。

### 混沌珠（紧急回档道具）

| 项目 | 内容 |
|------|------|
| 效果 | 疑心值直接降至当前心态区间下限（察觉→50，疑惑→25，无知→0） |
| 冷却 | 40楼 |
| 价格 | 待定 |
| 适用阶段 | 仅屈辱前（前半程），屈辱后无效（后半程无坏结局，不需要） |

**定位：** 濒死保命工具。疑心值快到70愤怒线时，混沌珠一拍回到区间底部，给玩家喘息空间重新布局。与定心符的区别：定心符是-8微调常规减压，混沌珠是一次性大幅回档救命。

**设计意义：** 150楼最多用~3次（冷却40楼），不会被滥用，但关键时刻能救一命，避免玩家因一次失误直接坏结局。

### 神魂共鸣石神魂空间效果

**基础效果（任何场景）：** 信任+5

**神魂空间额外效果（方式2注入）：**
- 脚本检测：本轮使用了神魂共鸣石 + `_当前互动模式 === '神魂空间'`
- 状态快照中追加提示：「神魂共鸣石生效中——云霜凝与{{user}}产生了短暂的灵魂共鸣，她会在{{user}}的软磨硬泡下勉强同意当前请求，但对自己为什么会答应感到困惑。仍受当前阶段行为限制。」
- 现实中使用：仅信任+5，无额外效果

### 蚀心露隐藏效果（组合道具）

**显性效果（每次使用）：** 防线-3, 信任+3, 疑心+3

**隐藏效果（一次性触发）：**
- 前置条件：苗广心态=「察觉」+ 隔音灵阵使用中 + 未曾触发过
- 效果：苗广心态→「屈辱」，疑心值重置为0
- 触发方式：**方式3**（立即触发AI回复）
- 道具说明不提示，纯脚本暗检测

**设计意义：**
- 这是触发苗广绿帽癖路线（察觉→屈辱→默许→沉溺）的**唯一方式**
- 不使用此组合 → 苗广疑心值只会一路涨到愤怒 → 坏结局
- 疑心值重置为0，后半程改名绿帽值从0起步
- 玩家仍需用定心符持续控制疑心值

## 五、三把锁重设计（回退道具）

原"锁定不升不降"改为"缓慢回退至初始值"，核心玩法是制造**认知失调反差感**。

| 道具 | 新效果 | 回退目标 |
|------|--------|---------|
| 寒心锁 60灵石 | 信任每轮-2 | 回退至5（初始值） |
| 破心锁 60灵石 | 防线每轮+3 | 回退至100（初始值） |
| 断情锁 80灵石 | 完成度每轮-0.5 | 完成度最低至21（阶段最低退至3） |

### 规则
- 回退**无视楼层天花板**（唯一能反向突破的道具）
- 可回退至初始值，不设回退上限
- **断情锁特殊规则：完成度可退至21（阶段最低退至3），不会退回植物人状态**
  - 效果：治疗成果流失，身体状态（寒毒）恶化，阶段可能随之倒退
  - 反差感："我明明已经好转了，为什么身体又开始变差"——恐惧、焦虑、更依赖{{user}}
- 卸下后立即停止回退
- 三把锁可同时使用，玩家自由选择
- 装备/服装/体改/性癖不受影响，保持当前状态
- **道具解锁条件只在购买时检查一次**，购买后永久拥有，装备/卸下不再检查条件。回滚数值后所有已购买道具照常使用——这是三把锁反差玩法的核心前提

### AI提示词动态注入
当检测到"锁使用中 + 数值正在回退"时，需注入反差描写提示。

**核心原则：不是"无法脱下/摘除"，而是"已经习惯了"**

按道具类别的描写指导：
- **服装类**：已习惯穿着，即使内心重新觉得羞耻也不会主动更换。可能某一刻突然意识到"我什么时候开始觉得穿这个正常了？"
- **性具类**：身体已适应存在感，摘除反而会不适。身体的记忆和心理的排斥在冲突
- **性癖类**：反应已刻入本能，理智上排斥但身体会自动响应。"明明不想要，但身体已经学会了"
- **体改类**：永久改变，即使心态回退也无法逆转。每次照镜子都会被提醒"自己已经不同了"

反差感的核心是**身体/习惯回不去了，但内心在挣扎**，而非系统强制

## 六、服装系统

### 基本规则

- **5个槽位**：上装、下装、内衣、内裤、特殊配饰
- **5个暴露等级**：Lv1微露 → Lv5极露
- 每件服装有**隐藏暴露分数**，5槽位总分决定当前暴露等级
- 暴露等级越高，苗广阶段推进值越大（高暴露是苗广阶段跳转的**硬性条件**之一）
- 服装受**治疗阶段门槛**解锁，解锁即代表角色心理可接受该暴露度
- 同级服装分数不同，全选高分款可跨级——玩家通过搭配探索发现

## 十一、苗广心态推进机制

### 总览：7种心态，双阶段驱动

同一变量 `苗广.疑心值`（0→100）全程复用，触发屈辱后重置为0，前端显示名从"疑心值"切换为"绿帽值"。

```
无知 → 疑惑 → 察觉 ─┬→ 屈辱 → 默许 → 沉溺（好路线）
                      └→ 愤怒 → 坏结局（坏路线）
```

### 前半程：疑心值（风险管理）

玩家需压住疑心值避免愤怒，适时引导进入屈辱。

| 疑心值 | 心态 | 苗广行为 |
|--------|------|---------|
| 0~25 | 无知 | 安心守候，不知情 |
| 25~50 | 疑惑 | 感觉有异，暗中观察，可能打断治疗 |
| 50~70 | 察觉 | 确认有异，分叉点——蚀心露+隔音灵阵组合可触发屈辱 |
| 70+ | 愤怒 | 坏结局，不可逆 |

**触发时机：** 脚本检测到云霜凝数值变化（信任↑/防线↓/身体开发↑/完成度↑）时，判定为一次治疗互动，触发疑心值增长。数值无变化则不触发，无需判断互动类型。

**疑心值增长源（每次治疗互动触发）：**

| 来源 | 疑心值/次 | 备注 |
|------|----------|------|
| 无遮蔽治疗（不装任何环境道具） | +5 | 声音泄漏+灵力异常 |
| 仅隔音灵阵 | +2 | 减弱但无法完全消除灵力波动 |
| 蚀心露 | +3/次 | 消耗品额外叠加，冷却25楼 |
| 服装暴露Lv2 | +1 | 苗广进房查看时注意到 |
| 服装暴露Lv3 | +2 | |
| 服装暴露Lv4+ | +3 | |

**疑心值减少源：**

| 来源 | 效果 |
|------|------|
| 定心符 | -8/次，冷却20楼 |

**打断治疗机制（屈辱前）：**
- 每轮脚本自动判定是否打断（治疗阶段1免疫）
- 基础概率按治疗阶段递增：阶段2=8%，阶段3=15%，阶段4=20%，阶段5+=25%
- 疑心加成：疑心值25~50 → +5%，疑心值50+ → +10%
- 硬上限35%（无论如何不超过）
- 隔音灵阵装备中：概率×0.5
- 神魂空间中：概率×0.5
- 打断 = 写入 `_待发送道具事件`（苗广闯入提示词），玩家下次发消息时附加，AI自然描写打断场景，当天治疗结束，{{user}}被赶走

**察觉→屈辱跳转（一次性）：**
- 条件：苗广心态=察觉 + 隔音灵阵装备中 + 使用蚀心露
- 效果：心态→屈辱，疑心值重置为0，前端改名绿帽值
- 详见第四节「蚀心露隐藏效果」

### 后半程：绿帽值（主动推进）

触发屈辱后，玩家从防守转为进攻，主动推进苗广堕落。

| 绿帽值 | 心态 | 苗广行为 |
|--------|------|---------|
| 0~40 | 屈辱 | 知道真相但无法接受，偷看时充满自我厌恶，不再打断治疗 |
| 40~75 | 默许 | 不再抗拒，合理化窥视行为，留影石可出售（500灵石） |
| 75+ | 沉溺 | 主动求看，期待，自慰，留影石主动求购（800灵石） |

**环境道具被动增长（每次治疗互动触发，道具装备中）：**

| 组合 | 绿帽值/天 | 苗广感知 |
|------|----------|---------|
| 都不装 | +1 | 他知道但看不到听不到 |
| 仅隔音灵阵 | +0 | 什么都感知不到 |
| 仅影绰纱帘 | +4 | 看到影子+听到声音 |
| 隔音灵阵+影绰纱帘 | +3 | 只看到影子 |
| 仅透灵幔 | +6 | 看清一切+听到声音 |
| 隔音灵阵+透灵幔 | +5 | 看清一切但听不到 |

**服装暴露叠加加成（与环境道具叠加）：**

| 暴露等级 | 额外绿帽值/天 |
|---------|-------------|
| Lv1 | +0 |
| Lv2 | +1 |
| Lv3 | +2 |
| Lv4 | +3 |
| Lv5 | +4 |

**净灵铃：** +5/次（方式3即时触发，每天限用1次）
